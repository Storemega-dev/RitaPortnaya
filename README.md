# Швейная мастерская "Мастер Рита"

Сайт швейной мастерской с возможностью просмотра услуг и расценок.

## Основные функции

- Каталог услуг (пошив, ремонт, модернизация одежды)
- Прайс-лист с актуальными расценками
- Простая админ-панель для редактирования контента
- Галерея работ и примеров
